# AI Usage

I used AI primarily for creating and refining tests and for refactoring. The main logic and functionality were implemented by me, with AI used as a tool to improve code quality, structure, and handling of edge cases.

Assumption: When a reserved book is returned, the reservation queue is processed in order. Members are removed from the front of the queue until the first eligible member is found, who then receives the book. Remaining members stay in the queue.
